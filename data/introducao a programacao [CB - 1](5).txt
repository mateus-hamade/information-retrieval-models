Introducao a programacao - Conceitos Basicos

Analise o codigo abaixo escrito em C. 

int main() {
    int a = 0, b = 1, c = 2;

    printf(“%d”; a);

    c = a;

    printf(“%d”; b);

    b = c;

    printf(“%d”; c);

    return
}

Assinale, a seguir, o valor impresso ao final da execucao do codigo.

B. 010;
