Algoritmos e Estrutura de Dados Estrutura de Dados , Filas , Pilhas

Pilhas e filas são duas estruturas de dados bem utilizadas até os dias de hoje. É correto o que se lê nas alternativas abaixo, EXCETO em:

Alternativas:
A. É possível implementar uma fila usando duas pilhas.
B. Pilhas seguem uma política LIFO – o último a entrar será o primeiro a sair.
C. Não há operações na pilha para verificar o elemento no topo da pilha sem removê-lo.
D. É possível implementar filas que realizem operações de entrada e saída de elementos em O(1).
E. É possível implementar uma operação que retorne a quantidade de elementos de uma pilha ou uma fila em O(1).
