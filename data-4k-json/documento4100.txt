Sistemas Operacionais Windows

O sistema operacional Windows usa um sistema de arquivos baseado numa abstração em letra para que o usuário possa distinguir entre um disco e outro (ou uma partição e outra). A respeito disso, é correto afirmar, EXCETO: 

Alternativas:
A. Derivado do sistema DOS onde reserva-se os drives A e B para disquetes, e o drive Com para disco rígido.
B. Os arquivos de cada disco ou partição são organizados a partir de um diretório raiz correspondente à sua letra.
C. Atualmente NTFS é utilizado como padrão.
D. Inicialmente foi desenvolvido o sistema FAT (12, 16, 32 e exFAT).
E. Apenas drivers de rede não podem ser mapeados para letras.
