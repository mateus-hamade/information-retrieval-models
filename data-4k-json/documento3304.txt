Algoritmos e Estrutura de Dados Estrutura de Dados , Árvores

Sobre árvores binárias balanceadas, considere as seguintes afirmativas:   1) São estruturas de dados onde cada nodo tem exatamente dois nodos filhos.  2) São estruturas de dados onde cada nodo tem no máximo dois nodos filhos.  3) São aquelas que equilibram a distância entre o nodo raiz e todos os nodos folha.   Assinale a alternativa correta.  

Alternativas:
A. Somente a afirmativa 1 é verdadeira.
B. Somente a afirmativa 2 é verdadeira.
C. Somente a afirmativa 3 é verdadeira.
D. Somente as afirmativas 2 e 3 são verdadeiras.
E. As afirmativas 1, 2 e 3 são verdadeiras.
