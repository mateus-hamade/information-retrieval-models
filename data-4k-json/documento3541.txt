Algoritmos e Estrutura de Dados Algoritmos , Lógicas de Programação

A lógica de programação é necessária para o desenvolvimento de sistemas e programas. Ela permite definir a seqüência lógica para o desenvolvimento. Com relação à lógica de programação, marque a alternativa correta. 

Alternativas:
A. Lógica de programação é a técnica de encadear pensamentos para atingir determinado objetivo.
B. Seqüência Lógica são passos executados antes de atingir um objetivo ou solução de um problema.
C. Um algoritmo é formalmente uma seqüência infinita de passos que levam a execução de uma tarefa.
D. Para escrever um algoritmo precisamos descrever a seqüência de instruções de maneira complexa e objetiva.
E. O diagrama de blocos é uma forma não padronizada e eficaz para representar os passos lógicos de um determinado processamento.
