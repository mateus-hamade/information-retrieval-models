Algoritmos e Estrutura de Dados Estrutura de Dados , Filas , Listas

Analise as afirmativas:  I. Java possui classes nativas para implementação de matrizes, listas, pilhas e filas. Isso aumenta a produtividade e reduz a dependência de pacotes de terceiros. II. Listas e filas são estruturas de dados similares. O que as diferencia é o protocolo de adição e remoção de elementos. Listas podem ter elementos inseridos ou removidos de qualquer posição, enquanto, nas filas, os elementos são inseridos no final e removidos do início. III. Além das listas, Java disponibiliza uma estrutura chamada “array”. Essa estrutura armazena dados de um mesmo tipo. Diferente de uma lista, que pode ter seu tamanho alterado, um array tem seu tamanho definido na sua criação. Um array que possui outros arrays em suas posições é chamado de array multidimensional. IV. Filas são implementadas utilizando-se o protocolo FIFO (first-in-first-out), ou seja, o primeiro a entrar é o primeiro a sair.
