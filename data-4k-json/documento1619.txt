Algoritmos e Estrutura de Dados Algoritmos , Algoritmos de Ordenação

Assinale a alternativa que descreve corretamente a operação que o método abaixo realiza sobre um vetor (v) de números inteiros.                                                        public void operacao(int[] v) {                                                        for(int i = 0; i < v.length - 1; i++) {                                                          for(int j = 0; j < v.length - 1 - i;                                                    j++) {                                                            if(v[j] < v[j + 1]) {                                                               int aux = v[j];                                                               v[j] = v[j + 1];                                                               v[j + 1] = aux;                                                             }                                                          }                                                       }                                                    }
