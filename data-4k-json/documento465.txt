Banco de Dados SQL

Observe o seguinte script SQL, executado em uma instância do H2 Database 2.2, com configuração padrão:  INSERT INTO CAMPOS (id, campo) VALUES (1, '2023-12-31 23:59:59Z'); SELECT campo FROM CAMPOS WHERE id = 1; Após a execução do script, o H2 Database 2.2 retornou: 2023-12-31 20:59:59  Logo, o tipo de dado do atributo campo da tabela CAMPOS no H2 Database 2.2 é:
