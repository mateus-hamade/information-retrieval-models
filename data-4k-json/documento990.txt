Sobre modelagem de dados, marque V para as afirmativas verdadeiras e F para as falsas. ( ) O grau de um tipo de relacionamento é o número de entidades fracas participantes. Relacionamentos de grau dois são chamados de binários; os de grau três são conhecidos como terciários. ( ) Os tipos de entidade que não possuem atributos-chave próprios são denominados tipos de entidade fraca. ( ) A agregação é utilizada para representar um relacionamento entre um objeto inteiro e suas partes componentes, possuindo uma notação diagramática distinta.
