Banco de Dados SQL

Assinale a alternativa que corresponde, na linguagem SQL, a uma sintaxe correta de uma instrução SELECT. 

Alternativas:
A. SELECT <nome(s)_colunas> WHERE <condição> GROUP BY <nome(s)_colunas> FROM <nome_tabela>
B. SELECT <nome(s)_colunas> FROM <nome_tabela> GROUP BY <nome(s)_colunas> ORDER BY <nome(s)_colunas> WHERE <condição>
C. SELECT <nome(s)_colunas> FROM <nome_tabela> WHERE <condição> GROUP BY <nome(s)_colunas> ORDER BY <nome(s)_colunas>
D. SELECT <nome(s)_colunas> ORDER BY <nome(s)_colunas> FROM <nome_tabela> WHERE <condição>
E. SELECT <nome(s)_colunas> WHERE <condição> FROM <nome_tabela> ORDER BY <nome(s)_colunas> GROUP BY <nome(s)_colunas>
