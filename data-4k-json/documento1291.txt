Algoritmos e Estrutura de Dados Estrutura de Dados

O técnico José implementou uma estrutura de dados linear na qual os elementos são organizados de modo a constituir uma sequência lógica. Na estrutura implementada por José, os elementos não são, necessariamente, armazenados de forma contígua na memória do computador, e o primeiro elemento a entrar é o último a sair. B. fila com alocação encadeada;
C. pilha com alocação sequencial;
