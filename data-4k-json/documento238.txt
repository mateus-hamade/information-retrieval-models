Banco de Dados SGBD - Sistema de Gerenciamento de Banco de Dados , PostgreSQL

Sistemas Gerenciadores de Bancos de Dados, SGBD, via de regra, permitem acesso às suas funcionalidades por meio de interfaces interativas e, também, por meio de programas de apoio ou comandos específicos. O SGBD PostgreSQL possui um conjunto de programas de apoio. O programa a ser utilizado para realizar o backup de um banco de dados ou de um esquema, no PostgreSQL, é o pg_dump. Nesse contexto, considere o seguinte comando:  pg_dump -a -Fp Alfa > Beta Esse comando gera um backup 

Alternativas:
A. das definições (esquema) do banco de dados Alfa, apenas
B. das definições (esquema) dos bancos de dados Alfa e Beta, em formato proprietário, apenas
C. de todo o banco de dados Beta (esquema e dados) em formato proprietário
D. dos dados do esquema Beta do banco de dados Alfa em formato textual (comandos SQL)
E. dos dados do banco de dados Alfa em formato textual (comandos SQL)
