Sobre o sistema de arquivos NTFS (New Technology File System), é CORRETO afirmar: 

Alternativas:
A. em discos de 2GB a 2TB o cluster padrão é de 4KB. É possível definir o tamanho do cluster em até 64KB no momento da formatação (mas não suporta a compressão de arquivos esparsos).
B. normalmente, os softwares escrevem em discos óticos um sistema de arquivo NTFS em um único passo.
C. há 3 tipos principais: Plain, VAT (virtual allocation table), e Spared (RW).
D. o uso de múltiplas sessões também é permitido, mas gera muitas vezes problemas de leitura e armazenamento.
E. usa o sistema LogFS, que é projetado para dispositivos de memória ash com grande capacidade.
