Algoritmos e Estrutura de Dados Algoritmos , Tipos de Dados

Em relação a tipos de dados, atribua V (verdadeiro) ou F (falso) às afirmativas a seguir. ( ) Char é um tipo simples. ( ) Double é um tipo estruturado. ( ) Float é um tipo simples. ( ) Int é um tipo simples. ( ) Void é um tipo estruturado. Assinale a alternativa que contém, de cima para baixo, a sequência correta. 

Alternativas:
A. V, V, F, F, F.
B. V, F, V, V, F.
C. F, V, V, F, V.
D. F, F, V, F, V.
E. F, F, F, V, V.
