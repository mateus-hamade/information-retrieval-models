Banco de Dados SGBD - Sistema de Gerenciamento de Banco de Dados

Um dos objetivos primordiais de um Sistema de Gerenciamento de Banco de Dados (SGBD) é a manutenção da integridade de dados sob seu controle. Para tentar garantir a integridade de um banco de dados, os SGBDs oferecem o mecanismo de restrição de integridade. Na abordagem relacional, uma restrição de integridade é uma regra de consistência que normalmente é garantida pelo próprio SGBD.
