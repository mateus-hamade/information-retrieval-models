Banco de Dados SGBD - Sistema de Gerenciamento de Banco de Dados

Um SGBD que está junto com o banco de dados em um servidor de banco de dados e que oferece acesso concorrente a vários usuários é classificado como um SGBD:

Alternativas:
A. Distribuído.
B. Centralizado.
C. Hierárquico.
D. NSQL.
E. SQL.
