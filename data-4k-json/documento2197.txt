Uma transformação polinomial é uma ferramenta fundamental na demonstração de que determinado problema é NP-difícil.     Avalie as afirmações sobre propriedades que transformações polinomiais devem satisfazer.     I. Para toda transformação polinomial, deve existir uma Máquina de Turing determinística que a computa em tempo polinomial.  II. Se uma transformação polinomial transforma um elemento de linguagem A em um elemento de linguagem B, então A é um subconjunto não necessariamente próprio de B.  III. Se uma transformação polinomial transforma um elemento de uma linguagem A em um elemento de linguagem B, e A pertence a NP, então B pertence a NP.  IV.
