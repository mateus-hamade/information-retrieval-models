Algoritmos e Estrutura de Dados Algoritmos , Lógicas de Programação

Considere a seguinte expressão lógica, a ser inserida em um programa.     R ← ((A and B) or (C and D)) or E     As variáveis A, B, C, D, E e R são do tipo booleano, podendo assumir valores verdadeiro (1) ou falso (0).     Assinale a alternativa que apresenta valores para A, B, C, D e E que resultem o valor falso (0) em R. 

Alternativas:
A. A=1 ; B=1 ; C=0 ; D=0 ; E=0.
B. A=1 ; B=0 ; C=1 ; D=1 ; E=1.
C. A=1 ; B=0 ; C=1 ; D=0 ; E=1.
D. A=0 ; B=1 ; C=1 ; D=1 ; E=0.
E. A=0 ; B=1 ; C=0 ; D=1 ; E=0.
