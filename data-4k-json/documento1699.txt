Algoritmos e Estrutura de Dados Algoritmos , Algoritmos de Ordenação

Assinale a alternativa CORRETA. Muitos algoritmos de ordenação de números armazenados em array necessitam fazer comparações (verificar se um número é menor, igual ou maior que outro número) e trocas (mudar a posição dos números dentro do array). Considere o problema de ordenar um array contendo N números inteiros. Se for utilizado o Método da Bolha, qual é o menor número possível de trocas para ordenar completamente o array? 

Alternativas:
A. N
B. 0 (zero)
C. N / 2 (metade de N)
D. 2 x N (duas vezes N)
E. N! (fatorial de N)
